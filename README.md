# 'Let’s find our neighborhood!' 
### 아파트 정보부터 우리 동네 정보까지
선택한 지역의 건물 정보 및 시세 정보를 제공합니다.<br>
회원가입 시 입력한 주소 기반으로 우리동네 게시판을 이용할 수 있습니다.

## 주요기능
- 선택한 지역의 건물 정보와 시세 정보 제공
- 회원가입 시 입력한 주소 기반으로 우리동네 게시판 생성
- 공지사항 및 FAQ 게시판
- 회원가입 및 회원정보 수정 기능
